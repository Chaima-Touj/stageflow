# 🌐 StageFlow Web App
---

## 👩‍💻 About the project
StageFlow is a web application that helps students, companies, and administrators manage internships easily.

---

## ⚙️ Features
- 👨‍🎓 Student can apply for internships
- 🏢 Companies can publish offers
- 📊 Admin dashboard for management
- 🔐 Authentication system (login/register)
- 📅 Internship tracking system

---

## 🛠️ Tech Stack
- Frontend: React / Vite
- Backend: Node.js / Express
- Database: MongoDB
- Authentication: JWT
